# IPL Score Predictor

This project predicts the final score of an IPL innings using machine learning.

## Dataset
IPL match data containing ball-by-ball information.

## Features
- Batting Team
- Bowling Team
- Current Runs
- Wickets
- Overs
- Runs in last 5 overs
- Wickets in last 5 overs

## ML Models Used
- Linear Regression
- Random Forest
- Decision Tree

## Tech Stack
Python  
Pandas  
NumPy  
Scikit-learn  
Matplotlib  

## How to Run

1 Clone repository

git clone https://github.com/Areefbaba/ipl-score-predictor

2 Install requirements

pip install -r requirements.txt

3 Run notebook

Open `ipl_score_prediction.ipynb`
